~~~ SAMBA Script Post-Install ~~~

I added an “Alias” that can be run to install and configure SAMBA, should you need it. Just run “smb” command in Terminal, follow instructions, making sure username you select is the same as you current one, then reboot.

There are tons of other aliases, to see them all, type “alias” in Terminal, or open the file .bashrc found in your home directory.

Finally, If you are new to Linux or not ready to do a bit of research then, this Distro is not for you.

My Calamares Config Explained

I made package list Dynamic, so it will be constantly changing, meaning that if you install it today, and I add/change a few things installing it in the future will present you with a different list, much the same but with a few packages removed or added, I did it this way so I don’t have to re-compile and re-upload ISO as I do not have enough bandwidth to spare, but for now I will try to explain below some of the sections below :

- Core : This is the main section, where you select Kernel and GPU Drivers, be careful though, if you select wrong GPU drivers for your Hardware you might end up with the infamous "Black Screen/No Desktop" issue. Do Your Research Beforehand. It's ESSENTIAL..

Note : Don’t forget to select your Processor’s Microcode, it’s very important !

- Apps : Here you can choose what apps you want out of the box. Many to select from. All very useful. I don’t put useless stuff.

- Gaming : If you want games, here’s where you select Launcher of choice with some “Native” games. Note if you select games it will make installation take much longer.

- Extras : This is where you can select a few more stuff I consider Extras. Like some custom tools by ArcoLinux and some themes.

~~~ Common Issues ~~~

No Internet Connection

Some users out there have reported an issue where Calamares installer was showing a message saying it’s wasn’t able to establish a connection to the Internet.

I have no idea why it happens or how to fix it beyond that… Will have to wait and see, maybe it'll be fixed in next version of Calamares :(

nVidia GPUs

Now it’s a known fact that nVidia don’t play well with Linux, so if you have an old Laptop/Desktop with an older GPU or Hybrid Intel/nVidia, you might get stuck with No Desktop, which means you didn't do enough reserach on the subject and selected wrong Drivers.

I included all the Drivers I know of in the list, it's up to you Do Your Research Before attempting to install... Best of luck on your journey ;)
