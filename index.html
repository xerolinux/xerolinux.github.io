Installation

First Steps

Now that we have established that this Distro was made for myslef primarily exercising my knowledge thus far, and that minimal support is available as I continue my endeavour to learn more as I go, learning is fun. Enjoy...

An Active Internet Connection Is Required. otherwise nothing will show and you will see empty lists.

I added an “Alias” that needs to be run on first logon to fix a bug with GRUB - just run “grubfix” command in Terminal then reboot. There are tons of other aliases - type “alias” in Terminal to see them all.

Very Important : Make sure to READ ENTIRE PAGE before proceeding. What works for me might not work for you. If you are new to Linux or not ready to do a lot of research then DON'T GET MY DISTRO

Ok, now on to the info, I opted to include PARU - the AUR - Helper in my Distro. I think it's better maintained, and more scalable as a Helper. You could install other Helpers if you so wish, but all Aliases - in my Distro use PARU -, so if you opt to use another one you will need to update them accordingly.

And I also included the ArcoLinux Repositories for some extra packages. They have a good selection of apps and tweaks. Always have a Backup before downloading packages you know nothing about, it's safest route to take.

Package List Explained

The list is Dynamic so it will be constantly changing, meaning that if you install it today, and I add/change a few things installing it in the future will present you with a different list, much the same but with a few packages removed or added, I did it this way so I don’t have to re-compile and re-upload ISO as I do not have enough bandwidth to spare, but for now I will try to explain below some of the sections below :

- Core : This is the main section, where you select Kernel and GPU Drivers, be careful though, if you select wrong GPU drivers for your Hardware you might end up with the infamous "Black Screen/No Desktop" issue. Do Your Research Beforehand. It's ESSENTIAL..

Note : Don’t forget to select your Processor’s Microcode, it’s very important !

- Apps : Here you can choose what apps you want out of the box. Many to select from. All very useful. I don’t put useless stuff.

- Gaming : If you want games, here’s where you select Launcher of choice with some “Native” games. Note if you select games it will make installation take much longer.

- Extras : This is where you can select a few more stuff I consider Extras. Like some custom tools by ArcoLinux and some themes.

~~~ Common Issues ~~~

No Internet Connection

Some users out there have reported an issue where Calamares installer was showing a message saying it’s wasn’t able to establish a connection to the Internet. In that case, since am not yet savvy enough to help, you will need to figure out how to fix via Google before going through with installation.

I have no idea why it happens or how to fix it beyond that… Will have to wait and see, till then install without selecting anything from list :(

nVidia GPUs

Now it’s a known fact that nVidia don’t play well with Linux, so if you have an old Laptop/Desktop with an older GPU or Hybrid Intel/nVidia, you might get stuck with No Desktop, which means you didn't do enough reserach on the subject and selected wrong Drivers.

I included all the Drivers I know of in the list, it's up to you Do Your Research Before attempting to install. As mentioned before, I am still in the early stages of my Linux adventure, so I am ill-equipped to help... Best of luck on your journey ;)
